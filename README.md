# Carers Community Circle

A safe, welcoming space for paid and unpaid carers to feel seen, supported, and understood.

This repository contains the static web platform for **Carers Community Circle**, a Community Interest Company (CIC) dedicated to bringing carers together across Enfield, Hackney, and Waltham Forest.

## What We Offer
- **Community & Connection**: Coffee mornings, discussion groups, and peer support.
- **Wellbeing & Activity**: Walks, exercise sessions, and outdoor activities.
- **The Knowledge Exchange**: Peer-led discussions on navigating services and everyday coping strategies.
- **Dementia-friendly screenings & Outings**: Group outings, relaxed cinema mornings, and museum trips.
- **Workshops & Learning**: Vocations, confidence building, digital skills, and CPD.

## Features of this Site
- 📱 **Mobile-First & Responsive**: Designed for carers who browse on the go.
- ♿ **Accessible & Semantic**: Optimized structure following WCAG contrast guidelines.
- ☕ **Interactive Event Invitation**: Hero section features a built-in interactive card detail for the upcoming coffee morning.
- 🔐 **GDPR & Privacy Compliant**: Features privacy/safeguarding policies and cookie settings.

## Technical Setup
The platform is built using pure static HTML5, modern vanilla CSS, and clean Javascript (no heavy libraries or external dependencies). Ready for lightweight static hosting.

### Local Development
To serve locally, you can start any static HTTP server in the root directory:
```bash
npx http-server -p 8000
```
Then visit `http://localhost:8000`.
